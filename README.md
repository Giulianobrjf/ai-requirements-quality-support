# AI Requirements Quality Support

This project is a proof of concept developed as part of an undergraduate thesis.
It explores the use of generative AI to support the creation and quality analysis of software requirements.

## Technologies
- Python
- FastAPI
- React
- Generative AI via API

## Scope
The system supports:
- Creation of software requirements from user stories
- Quality analysis and improvement of existing requirements

This project focuses on prompt engineering and qualitative analysis, not on model training.
